sshs¶

pylayers.antprop.antssh.sshs(G, fGHz, th, ph, L=20)[source]¶

scalar spherical harmonics transform

G : antenna gain (f,th,phi) fGHz : np.array th : np.array ph : np.array

This function calculates the Scalar Spherical Harmonics coefficients

m : phi longitude l : theta latitude

Antenna pattern are stored (f theta phi) Coeff are stored with this order (f , l , m )
